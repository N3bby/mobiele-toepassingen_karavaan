EvenExpense

EvenExpense

An EvenExpense is an expense where the debt gets evenly divided by participant. First, the total expenseAmount needs to be set, then it is possible to add payments and after that it is possible to add participants for debt calculation. It is not possible to add BillItems to an EvenExpense, only add payments and participants. It is not possible to add Debts to an EvenExpense, only add payments and participants.
